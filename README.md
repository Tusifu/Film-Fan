# FILM FAN 

## Requirements

1. Flutter v2.10.3
2. Dart v2.16.1

## Get Started

1. Checkout to master branc
2. Clone the project.
3. run: flutter clean
4. run: flutter pub get
5. recommended start the app in Debug mode.
